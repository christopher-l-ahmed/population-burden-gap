# Δ Population Burden Gap (ΔPBG)
This repository contains the reproducible R code for calculating the Population Burden Gap (PBG) and its longitudinal derivative, ΔPBG.

I introduced the ΔPBG metric as a method to track change over subsequent versions of Environmental Justice (EJ) cumulative impact mapping datasets (using Washington State’s EHD Map as an example).
