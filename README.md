# Δ Population Burden Gap (ΔPBG)
This repository contains reproducible R code for calculating the Population Burden Gap (PBG) and its longitudinal counterpart, ΔPBG. The ΔPBG metric tracks how population‑weighted burden disparities change across successive versions of Environmental Justice (EJ) cumulative impact mapping datasets (using Washington State’s EHD Map as an example).
